🏦 AI RELATIONSHIP MANAGER FOR BANKS
================================

Complete working prototype with:
✅ Full-stack Node.js + Express + PostgreSQL
✅ Risk scoring engine (0-100 score system)
✅ AI-powered outreach message generation
✅ Interactive dashboard with filters
✅ 25 realistic customer seed data
✅ Complete API with 7 endpoints
✅ Modern, responsive UI (no frameworks)

QUICK START (5 MINUTES)
=======================

1. INSTALL DEPENDENCIES
   $ npm install

2. COPY ENVIRONMENT FILE
   $ cp .env.example .env

3. ENSURE PostgreSQL IS RUNNING
   Windows: Use Services or Task Scheduler
   macOS: brew services start postgresql
   Linux: sudo service postgresql start

4. SETUP DATABASE
   $ node scripts/setup-db.js

5. START SERVER
   $ npm start

6. OPEN BROWSER
   http://localhost:3000

✅ You're running!

RISK SCORING ALGORITHM
======================

Points added for:
• No transactions 10+ days           → +20
• Login frequency < 3/week           → +15
• Complaint count > 2                → +25
• Balance drop (< 50% monthly avg)   → +20
• No salary detected                 → +20
• Low balance < $5,000               → +15

Final Score Scale:
  0-30   → LOW (Monitor)
  30-70  → MEDIUM (Email + Incentive)
  70-100 → HIGH (Call + Special Offer)

TROUBLESHOOTING
===============

❌ "connect ECONNREFUSED 127.0.0.1:5432"
   → PostgreSQL not running
   → Windows: Start from Services
   → macOS: brew services start postgresql

❌ "database does not exist"
   → Run: node scripts/setup-db.js

❌ Frontend not loading
   → Ensure npm start is running
   → Check http://localhost:3000

❌ OpenAI errors
   → Leave OPENAI_API_KEY commented in .env
   → System will use mock responses automatically
